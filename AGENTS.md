# AGENTS.md — エージェント役割定義

## クロウ（OpenClaw）

- **役割**: オペレーター
- **担当**: ターゲット発見・文章生成実行・送信承認フロー・結果記録
- **できること**: Web検索・Gmail送信・did-event API呼び出し
- **できないこと**: コード実装・DBスキーマ変更

## Claude Code

- **役割**: 実装エージェント
- **担当**: 機能追加・バグ修正・スキーマ変更・テスト
- **必読ファイル**: CLAUDE.md → progress.md → 該当 rules/ → 該当 skills/
- **禁止**: 承認なしの送信実行・GitHub へのプッシュ

## Cursor

- **役割**: UI実装エージェント
- **担当**: コンポーネント・スタイル・フロントエンド
- **禁止**: ビジネスロジックの変更（src/lib/ 配下は Claude Code 担当）

## Shunさん（人間）

- **役割**: 戦略・最終承認
- **担当**: キャンペーン方針決定・送信承認・優先度判断
- **必須介入ポイント**: 送信実行前・新しいターゲット条件の追加前

## 共有メモリ

`progress.md` がすべてのエージェントの共有コンテキスト。
セッション開始時に必ず読み、終了時に必ず更新する。
